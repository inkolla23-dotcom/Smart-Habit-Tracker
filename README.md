# Smart Habit Tracker with Reward System 🌟
Smart Habit Tracker with Reward System 🌟 is a Python-based application that helps users build and maintain positive habits through daily tracking and rewards. It allows users to monitor progress, earn points for completed habits, receive motivational feedback, and improve consistency with an interactive and engaging system.

Smart Habit Tracker with Reward System is a Python-based application designed to help users build and maintain healthy daily habits through an engaging point-based system. The application allows users to add multiple habits and track their completion over several days. For every completed habit, users earn reward points, while missed habits reduce points, encouraging accountability and consistency.

The system provides instant feedback after each habit entry, helping users stay motivated throughout their progress journey. It also includes achievement milestones where users can unlock a "Level Up" reward after reaching a target score. Warning messages are displayed when progress drops below expected levels, helping users identify areas that need improvement.

At the end of the tracking period, the application generates a final habit report showing the total points earned and overall performance. This project demonstrates fundamental Python concepts including loops, lists, conditional statements, user input handling, and decision-making logic while creating a practical productivity and self-improvement tool.

## Features
- Add and track multiple habits
- Monitor daily habit completion
- Reward points system (+10 for completion)
- Penalty system (-5 for missed habits)
- Instant motivational feedback
- Progress warnings for low performance
- Achievement and Level-Up system
- Final performance summary report
- Interactive command-line interface
- Supports multiple tracking days

## Technologies Used
- Python
- Lists
- Loops
- Conditional Statements
- User Input Handling
